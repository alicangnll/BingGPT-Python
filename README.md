# BingGPT - Bing AI API on Python
You can use BingGPT AI API on Python!
# Requirements
<ul>
  <li>Python 3.8+</li>
</ul>
<h1>Installation</h1>
<ul>
  <li>pip install -r requirements.txt</li>
  <li>pip install dist/binggpt-0.0.18-py3-none-any.whl OR pip install bingpython</li>
</ul>
